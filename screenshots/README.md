# 截图目录

请在此目录放置项目运行截图：

## 推荐截图

1. **dashboard.png** - 前端看板截图
2. **api_test.png** - API 测试截图（curl 或 Postman）
3. **mcp_stdio_test.png** - MCP stdio 测试输出截图
4. **indexer_output.png** - 索引器运行输出截图
5. **verify_data.png** - 数据验证脚本输出截图

## 截图命令

Windows:
- 使用 Win + Shift + S 截图
- 保存到此目录

macOS:
- 使用 Cmd + Shift + 4 截图
- 保存到此目录

## 截图要求

- 格式: PNG 或 JPG
- 分辨率: 建议 1920x1080 或更高
- 内容: 展示功能运行结果
